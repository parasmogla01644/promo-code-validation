# Promo Code Validation System

A full-stack promo code validation system built with ReactJS and NodeJS (Express).  
Includes client-side validation, server-side validation, logging and error handling

---

## Tech Stack

- ReactJS
- NodeJS
- Express
- HTML/CSS
- Git
- Axios

---

## Setup Instructions

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
